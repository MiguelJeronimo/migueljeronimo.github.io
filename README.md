# Personal Portfolio - Resume Website

A professional resume website for a Systems Engineer, featuring a modern, dynamic design built with HTML, CSS, and vanilla JavaScript.

## Features

- **Modern & Dynamic Design**: Built with Vanilla JS, HTML5, and CSS3. No frameworks required.
- **Single Page Application**: Seamless navigation between sections without page reloads.
- **Responsive Layout**: Optimized for all devices, from mobile to desktop.
- **Animated Hero Section**: Eye-catching hero section with smooth text reveal.
- **Interactive Navigation**: Active section highlighting in the navigation bar.
- **Smooth Scroll**: Animated scrolling to different sections.
- **Social Links**: Quick links to LinkedIn and GitHub profiles.
- **Projects Showcase**: Featured projects with details and links.

## Getting Started

### Prerequisites

- A modern web browser that supports HTML5, CSS3, and ES6 JavaScript.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```
2. Open `index.html` in your browser:
   ```bash
   open index.html  # macOS
   # or
   start index.html # Windows
   ```

## Structure

The website is organized into several key sections:

- **Header**: Contains the logo and navigation menu.
- **Hero Section**: Main introduction with a call to action.
- **About Section**: Personal information and introduction.
- **Projects Section**: Showcases software projects with descriptions and links.
- **Contact Section**: Contact information and social links.
- **Footer**: Copyright information and quick links.
